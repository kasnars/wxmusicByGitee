## 微信小程序云音乐树洞  
### 所用技术栈  
- 微信原生小程序  
- nodejs微信云函数  
### 所实现的功能  
- 实现播放器功能，并实现歌词和进度条实时滚动  
- 实现简单树洞留言功能，并支持回复留言，实现留言模糊搜索
- 实现对近期播放保存的最近收听功能  
- 利用微信小程序api实现分享小程序二维码和分享小程序当前页面功能
